inferlo.testing.pairwise_model_on_graph¶

inferlo.testing.pairwise_model_on_graph(graph, al_size=2, zero_field=False, seed=0)[source]¶

Builds random pairwise model with given interaction graph.

Parameters

  • graph – Interaction graph. Nodes must be labeled with consecutive integers, starting with 0.

  • al_size – Alphabet size.

  • zero_field – Whether model should be zero-field.

  • seed – Random seed.

Returns

Generated model.
